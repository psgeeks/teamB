### 자료구조 설계
1. id의 범위가 10억 -> Map을 통해 인덱스로 해싱
2. LCA가 필요 -> parent, dist, depth 배열
3. 트리 구현 -> 인접리스트

### init
1. 만든 자료구조들을 초기화시켜준다.
2. 새로할당은 비효율적이므로 최대 크기로 만들어놓고 초기화만 실시

### connect
1. 노드에 인덱스 부여
2. 인덱스를 해싱
3. 부모와 연결하므로 parent, dist, depth 초기화

### measure
1. LCA를 구함
2. dist[a] + dist[b] - 2*dist[LCA(a,b)]

### test
1. 해당 노드를 루트로 정함
2. 자식 노드들의 서브트리에서 가장 긴 길이와 두 번째로 긴 길이를 구한다.
